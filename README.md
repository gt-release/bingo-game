# Neo Bingo

ローカルでそのまま開ける、グラフィックが現代的なビンゴゲームです。インストール不要で、`index.html` をブラウザで開くだけで遊べます。

## 特長
- 5x5 のビンゴカード（中央は FREE）。
- 番号抽選、番号リスト、オート再生 (1.5 秒間隔) を搭載。
- 抽選された番号はカード上で自動ハイライト。マニュアルでのマークも可能。
- ビンゴ達成時はステータスとトーストでお知らせ。
- ダークなネオン調の UI とレスポンシブレイアウト。

## 遊び方
1. `index.html` をブラウザで開く。
2. 「番号を引く」または「オート再生」で抽選。
3. 自動でマークされるので、タップで手動マークもできます。
4. ビンゴがそろうと "BINGO!" が表示され、トーストで知らせます。
5. 新しいカードが欲しいときは「新しいカードを作る」を押してください。

## 開発メモ
- 依存パッケージはありません。モダンブラウザで動作します。
- UI は `style.css`、ロジックは `app.js` にまとまっています。

### テスト
Node.js のビルトインテストランナーでロジックを検証できます。

```bash
npm test
```
